# Saved Places Example

Demonstrates the `trufi_core_saved_places` package.

## Getting Started

```bash
flutter pub get
flutter run
```

## Features Demonstrated

- Viewing saved places (home, work, favorites, custom)
- Adding new places
- Editing existing places
- Deleting places
- Viewing recent history
- Clearing history
- Localization support (EN, DE, ES)
